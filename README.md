# Wstęp do gita

Po wykonaniu danego ćwiczenia poprzedź je znakami (*).
Przykładowo ćwiczenie przed wykonaniem wyglądające następująco:
```
N. Ćwiczenie do wykonania.
```
po wykonaniu ma wyglądać następująco:
```
N. (*) Ćwiczenie do wykonania.
```
